html

htmlとは?

うえで説明したようにhtmlはサイト上に文章や画像を表示するためのものです。

でも文字をそのまま打ってもパソコンやスマホなどのコンピューターは思ったように表示してくれません。

そのために、コンピューターが理解できるように正しい文法で文字を書いていくことで自分の思い通り表示させる必要があります。

そのコンピューターが理解できる文法というのがhtmlなのです。つまりhtmlはコンピューター用の一つの言語なのです。

htmlの文法

さあ、それではhtmlの文法について説明していきましょう。

まずこの画像を見てください

この画像はこのサイトのhtmlの一部です。

なにやらいままで表示してきた文が「<h2>」や「<p>」で囲まれていますよね。 そう、これこそがhtmlの文法です。

詳しく見ていきましょう。「そもそもhtmlって?」は 「<h2>」と「</h2>」で囲まれています。この 「<h2>」と「</h2>」はタグといい、htmlを読むうえで最も重要なものです。

タグにはたくさんの種類があります。「h」というのは「Heading」の略で見出しという意味です。ここではh2ですがh1~h6までの種類があります。数字が大きいほど小さい見出しになります。

この見出しのタグで囲った文字はほかの文字より大きく、太く表示されます。

上の文で囲うといいましたが、<タグ名>内容</タグ名> と書くと内容の部分が囲われているとコンピューターが判断しタグに沿った表示の仕方をします。(囲わなくてもいいタグも存在する。)

htmlの文法は基本的にこれだけです。簡単でしょ!

css

cssとは?

さあ次はcssについて説明します。眠くなってきた?じゃあ自分がパソコンでポチポチ長い文を打っている姿を想像してください。かっこいいでしょ?わくわくしてきたでしょ?(僕だけ?)

じゃあそんな姿になれるように残りも頑張って読みましょう!(笑)

さ、気を取り直してcssはhtmlで表示する文章や画像などの見た目(色や表示する場所)を整えるためのものです。cssはhtmlとセットで使うのです。

そしてcssにもコンピューターに理解してもらうため文法があります。

cssの文法

それでは下の画像を見てください。

h1と{}がありその{}の中にいろいろと書かれていますね。

順番に説明していきましょう。

まずh1というのは、htmlのh1を参照しますよ～という宣言(?)のようなものです。そして{}の中に宣言したタグの変更することを書いていくのです。ちなみにここのh1という部分はセレクターといいます。

つぎに{}の中、左の青の部分を和訳すると、「上はテキストをそろえる」　下は「色」という意味です。勘のいい人ならもう気づいたでしょう。

左に何をするのか指定して、右に内容を書くのです。左を「プロパティー」、右を「値」といいます。

「プロパティー:値;」これがcssの文法です。
